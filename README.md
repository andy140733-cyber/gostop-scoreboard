# 고스톱 점수판 🌸

나·엄마·아빠 3인이 치는 고스톱의 점수를 기록·누적·정산하는 웹앱입니다.
설치가 필요 없고, 데이터는 이 기기 브라우저에 자동 저장됩니다.

## 어디서 작동하나요? (PC·아이폰·안드로이드·아이패드)
이 앱은 **순수 웹앱**이라 운영체제와 무관하게 **모던 브라우저(크롬·엣지·사파리)**에서 그대로 돌아갑니다. 별도 앱 출시(앱스토어·구글플레이)가 전혀 필요 없습니다. 또한 **PWA(설치형 웹앱)**로 만들어져 있어, 한 번 주소로 열면 **홈 화면에 설치**하고 **오프라인에서도** 쓸 수 있습니다.

### 실행 방법 1 — 이 PC에서 바로
`index.html` 더블클릭. (크롬·엣지 권장. 데이터는 그 브라우저에 자동 저장)

### 실행 방법 2 — 휴대폰·태블릿 등 다른 기기에서 (권장)
다른 기기에서 쓰려면 파일을 **웹에 한 번 올려 주소(URL)로 여는 방식**이 가장 편합니다. 무료로 가능합니다.

1. 무료 정적 호스팅에 이 폴더를 올립니다. 쉬운 순서대로:
   - **Netlify Drop** ([app.netlify.com/drop](https://app.netlify.com/drop)) — 폴더를 드래그&드롭하면 즉시 주소 발급 (가장 간단)
   - **GitHub Pages** — 깃허브 저장소에 올리고 Pages 활성화
   - **Cloudflare Pages / Vercel** — 폴더 업로드
2. 발급된 주소(예: `https://우리집고스톱.netlify.app`)를 휴대폰 브라우저에서 엽니다.
3. **홈 화면에 추가** 하면 앱 아이콘처럼 쓸 수 있고, 이후 인터넷이 없어도 실행됩니다.
   - 아이폰(사파리): 공유 → "홈 화면에 추가"
   - 안드로이드(크롬): 메뉴(⋮) → "홈 화면에 추가" / "앱 설치"

> 같은 와이파이라면, 이 PC에서 `python -m http.server 8123` 을 켜고 휴대폰에서 `http://이PC의IP:8123` 으로 접속하는 방법도 있습니다(PC가 켜져 있는 동안만).

## 데이터 저장과 기기 이전
- 기록은 **그 브라우저의 `localStorage`**에 자동 저장됩니다(기기·브라우저별로 독립). 자동 클라우드 동기화는 없습니다(단일 기기 방식).
- **다른 기기로 기록을 그대로 옮기려면** — **설정 → 데이터 백업·복원·기기 이전**에서:
  - **📋 텍스트로 복사** → 옮길 기기의 **📥 붙여넣기로 복원**에 붙여넣기 (카톡·메모 등으로 전달, 모바일에 가장 편함)
  - 또는 **⬇ 파일로 내보내기 / ⬆ 파일에서 가져오기** (JSON 파일)
- 복원하면 그 기기의 데이터를 가져온 데이터로 덮어씁니다.

## 점수 규칙 (이 집 하우스룰)
한 판을 기록하면 아래 규칙대로 세 명의 점수 변화가 자동 계산됩니다. **항상 합은 0** (딴 만큼 잃습니다).

- **승자 1명 / 점수(base)**: 누적의 기준 점수.
- **피박**: 패자별로 적용. 피박당한 패자는 점수를 **2배**로 잃습니다. (두 패자 모두 가능)
- **폭탄 · 쓰리고 이상 · 대판**: 각각 한 판의 점수를 **×2**. 여러 개가 겹치면 **곱연산**(예: 폭탄+대판 = ×4, 셋 다 = ×8).
- **독박**: 고를 외친 뒤 진 사람이 진 점수를 **전부 혼자** 부담합니다. (피박·배수는 금액 계산에 그대로 반영)

> 예) 내가 고 외친 뒤 엄마가 3점 승, 아빠 피박, 대판이면 → 나 −18, 아빠 0, 엄마 +18

## 주요 기능
- **＋ 한 판 기록**: 승자·점수·피박·배수·독박을 고르면 결과를 실시간 미리보기 후 저장. 날짜·시간 자동 기록.
- **누적 점수판**: 현재 회차의 세 명 누적 점수(1등 👑).
- **기록 탭**: 회차·유형·승자·기간·특수조건으로 필터링, 각 기록 수정/삭제.
- **통계 탭**: 현재 회차 / 통산 전체 / 지난 회차별 순위·승률·득실·특수조건 빈도.
- **점수 보정**: 잘못 계산된 점수를 수동으로 조정(사유 메모).
- **정산하기**: 현재 회차를 마감하고 누적을 0으로 초기화. 기록은 보존되어 나중에 조회 가능.
- **설정**: 점당 금액 환산, JSON 백업/복원, 정산 이력, 전체 초기화.

## 점수 엔진 자체검증
`tests/scoring-tests.html`을 브라우저로 열면 계산 규칙의 자동 테스트 결과(전체 통과 여부)를 볼 수 있습니다.

## 구조
빌드 도구 없이 순수 HTML/CSS/JS로 작성되었습니다.
```
index.html            화면 구조
css/styles.css        화투 테마 스타일
js/config.js          상수(플레이어·배수·저장 키)
js/scoring.js         점수 계산 엔진(순수 함수)
js/store.js           localStorage 저장/복원
js/records.js         기록 추가·수정·삭제
js/ledger.js          누적 계산·정산 회차 분할·필터
js/stats.js           통계 지표
js/money.js           점수→금액 환산
js/backup.js          JSON 백업/복원
js/ui.js              화면 렌더링·이벤트
js/app.js             시작점 + 서비스워커 등록
manifest.webmanifest  PWA 설치 정보
sw.js                 오프라인 서비스워커(앱 셸 캐시)
icons/                앱 아이콘(192·512·애플 터치)
tests/scoring-tests.html  엔진 자체검증
```

> 반응형: 휴대폰=세로 1열, 아이패드=널찍한 1열, 노트북·데스크톱=좌측 점수판 고정 + 우측 2단으로 자동 조정됩니다.
